## Dependencies

* Node (18.18.1)
* Typescript (5.2.2)
* NestJS (10.2.1)

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Architecture and design decisions

* Use only e2e testing which gives us the best confidence the application is working and enable easy refactoring.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
